# HumanUnity â€“ HCI Project (Interazione Uomo-Macchina)

HumanUnity is a web application designed to support students and university staff in dealing with racial/ethnic discrimination episodes in educational contexts.  
The system helps users **recognize an episode**, get **immediate guidance**, **document it safely**, and **submit a report** (either quick or detailed), with a traceable follow-up.

This project was developed following the course assignments for Human-Computer Interaction.

- Provide a simple and safe space to understand whether an episode is discrimination.
- Reduce the fear of reporting through anonymity and guided flows.
- Enable a **quick report (1 minute)** or a **detailed step-by-step report (wizard)**.
- Offer case tracking for users and follow-up tools for university referents.
